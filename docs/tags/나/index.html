Travel

19 May, 2014 (2) – San Jose, CA

언젠가부터 사람들과 잘 지내는 게 쉽지가 않다. 스무살 때 대학을 가기 위해 아는 사람 하나 없이 혼자 서울로 올 때도, 나는 혼자 떠나는 아들을 걱정하는 엄마에게 아들이 다른 건 몰라도 사람들이랑 친하게 지내는 거 하나 만큼은 정말 잘한다고, 그건 정말 아무 걱정 하지 말라고 큰 소리를 쳤던 기억도 난다. 실제로 잘 지냈다. 지금 생각해보면 조금 의아하기도 하지만 나름 새내기 대표를 해보기도 했다. 친한 사람은 제법 있으면서도 불편한 사람은 없었다. 2학년 때는 동아리까지 만들어 회장 노릇도 하고, 기숙사 학생회 활동도 하고, 다른 사람이 보기에나 내가 느끼기에나 나는 누구보다도 주위 사람들과 잘 어울리는 사람이었던 것 같다.
