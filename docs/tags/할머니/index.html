할리스에서

할리스에서 밤 늦게 혼자 일을 하고 있는데, 맞은 편 자리에 한 할머니가 앉으셨다. 처음에는 껌 등을 팔거나, 아니면 동냥을 하러 다니는 분일까 하는 생각이 얼핏 들었다. 입고 계신 옷이나 차림새가 누추한 수준까지는 아니었지만 그렇다고 귀부인으로는 전혀 보이지 않았고, 카페라는 곳에 몇 번 와보신 적이 없는지 어색하고 불안한 모습으로 카페 안을 서성이셨기 때문이다. 조금 뒤 한 쪽에 자리를 잡고 앉으셨다. 커피를 시키지는 않으셨는데, 아무래도 버스도 끊기고 차가 없는 시간이다 보니 추위를 피해 들어오신 것 같았다.
