커피

요즘들어 부쩍 커피를 많이 마신다. 생각해보니 하루에 세네잔은 꼬박꼬박 마시는 듯 하다. 원두커피는 몸에 좋다는데 나는 맥심 모카골드, 커피믹스만 마신다. 고등학교 때 까지는 머리 나빠진다는 말에 괜히 꺼림찍해서 거의 마시지 않았고, 대학교에 와서도 된장질 하겠답시고 카페를 가거나 할 때가 아니면 역시 입에 대지 않았었는데, 요즘은 하루 일과를 끝내고 집에 갈 때 내 책상 위 여기저기에 놓여있는 빈 종이컵들을 보며 깜짝 놀랄 정도다. 다름이 아니라 연구실에 항상 비치되어 있기 때문이다. 마시면 잠이 안온다는 커피를 무료로 제공하는 의도가 살짝 의심스럽긴 하나, 커피 1.
